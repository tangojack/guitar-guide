Guitar Guide

A songwriting aid to take your guitar playing to the next level

Project Information

Features

Harmony

Learn fundamental concepts of harmony and chord voicings all over the fretboard in an intuitive way

Melody

Interactive routines that teach important fretboard shapes.

Songwriting

Learn how to improvise and create songs without the help of a tutor.

Project Plan

Technologies used

Music Information Retrieval

Use Digital Signal Processing to recognize guitar notes

Computer Vision

Algorithms to detect hand position which will help in finding all chord voicings

Machine Learning

Use a model that is trained on existing songs to generate chord progressions
